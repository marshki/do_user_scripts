change-hostn.sh 
---------------

## This script: 

* assigns user input to the variable `newhostn`
* displays current hostname 
* displays change to be made 
* assigns `newhostn` to both `/etc/hosts` and `/etc/hostname`
* makes change permanent upon reboot 

in Ubuntu 14.04.
